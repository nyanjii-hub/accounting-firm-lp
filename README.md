## 制作のポイント

### 1. 士業向けの信頼配色をカスタムカラーで一元管理

ネイビー（`#12355b`）を軸にしたブランドカラーパレット `aoba-*` を `tailwind.config.ts` で定義し、全セクションで統一。  
「誠実・信頼・専門性」を視覚的に表現し、士業サイトとして違和感のないデザインにしています。

### 2. ヒーローのグラデーションマスクでPC・スマホ両対応

背景画像の上に `bg-white/82`（スマホ）と `md:bg-gradient-to-r md:from-white`（PC）を重ねることで、  
どの画面サイズでもテキストの可読性を確保しつつ、写真の雰囲気をそのまま活かしています。

### 3. 問い合わせ動線を3経路で設計

「初回相談申込（メール）」「電話相談」「サービス内容確認」の3つのCTAをヘッダー・ヒーロー・フッターの複数箇所に配置。  
ユーザーの行動パターンに合わせて離脱を防ぐ導線設計にしています。

## Setup

```bash
# 依存パッケージのインストール
npm install

# 開発サーバーの起動
npm run dev
```

ブラウザで `http://localhost:3000` を開いて確認できます。

```bash
# 型チェック
npm run typecheck

# ビルド確認
npm run build
```

## 注意事項

- 本リポジトリはポートフォリオ用のサンプルです。「青葉公認会計士事務所」は架空の事務所名です。
- ヒーロー画像は `public/images/aoba-office-hero.png` を参照しています。商用利用時は自事務所の写真に差し替えてください。
- 電話番号・メールアドレス・住所はサンプル値です。実際に公開する場合は必ず差し替えてください。
- 料金表はあくまでサンプルです。実際の業務量・面談頻度に応じて変更してください。
- お客様の声はサンプルテキストです。実際の口コミを掲載する場合は掲載許可を確認してください。
